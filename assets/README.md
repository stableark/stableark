# Brand assets

**Canonical mark:** the ark/ship bow from `og.png` (dark green on cream). All icons use that illustration—not the old geometric U-hull.

Palette: cream `#f7f5f0`, forest green `#0b3d2e`.

| File | Use |
| --- | --- |
| `og.png` | Open Graph / social preview (source look for the ship) |
| `mark.png` | Primary mark; site favicon / header |
| `avatar.png` | GitHub org/repo avatar (cream ship on green) |
| `logo.png` | Wordmark lockup (ship + “Stable Ark”) |
| `favicon-32.png` / `favicon-16.png` / `apple-touch-icon.png` | Same mark, for legacy link tags |

## GitHub setup (manual)

1. **Organization avatar:** upload `avatar.png`
2. **Repo social preview:** upload `og.png`
