# Brand assets

Palette: cream `#f7f5f0`, forest green `#0b3d2e`.

| File | Use |
| --- | --- |
| `mark.svg` | Primary mark (cream bg); site favicon |
| `mark.png` | Raster mark 512² |
| `avatar.svg` / `avatar.png` | GitHub org/repo avatar (green bg) |
| `favicon-16.png` / `favicon-32.png` | Legacy favicon PNGs |
| `apple-touch-icon.png` | iOS home screen |
| `logo.png` | Wordmark lockup (marketing) |
| `og.png` | Open Graph / GitHub social preview (~1536×1024) |

## GitHub setup (manual)

1. **Organization avatar:** Settings → Profile → upload `avatar.png` (or SVG if accepted).
2. **Repo social preview:** Repo → Settings → General → Social preview → upload `og.png`.
3. Site already references `og.png` via meta tags on [stableark.org](https://stableark.org) once Pages is updated.
