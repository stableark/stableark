Stable Ark

Self-custodial, dollar-denominated balances on Bitcoin using Ark VTXOs.

Status: research prototype — design note published; no production software yet.

Read the design note Source on GitHub

Stable Ark pairs a USD-indexed claim with a leveraged bitcoin counterparty. Position updates use Ark out-of-round transfers; rounds renew expiry and restore stronger refresh-VTXO security.
