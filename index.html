Stable Ark

Self-custodial, dollar-indexed bitcoin balances on Ark-family VTXOs. No issued token.

Status: research prototype — design notes published; Arkade-first PoC planned; no production software yet.

Read the design note User onboarding Source on GitHub

Stable Ark pairs a USD-indexed claim with a leveraged bitcoin counterparty on a normal public Ark operator. Price updates are joint multi-input offchain reallocations; lifecycle renewal uses each stack’s batch or intent path—not the price clock.
